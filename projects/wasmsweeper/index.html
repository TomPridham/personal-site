Wasmsweeper

A minesweeper clone written in Rust, using the library Bevy, compiled into WASM. The repo is here. It uses WebGL2, so it doesn't currently work in Safari. You might need to enable WebGL2 for it to work. If it's not working in Firefox, you might need to do: about:config => webgl.force-enabled = true

Click anywhere to get started. Right click mines to flag them
